## The Formula

```
C(t) = S(t) · A(t) · M(I)
```

| Symbol | Meaning                                            |
| ------ | -------------------------------------------------- |
| `C(t)` | Creative output over time                          |
| `S(t)` | Self-knowledge — how clearly you see yourself      |
| `A(t)` | Acceptance — how honestly you receive what you see |
| `M(I)` | Quality of what enters the system                  |

## The Error Function

```
ε = (1 − S)(1 − A)
```

Every blind spot multiplied by every resistance. Compounding silently.

## The Stability Condition

```
d(S · A)/dt  >  dε/dt
```

Growth must outpace drift. The moment it doesn't, the system produces confident wrong outputs.
